## Small Swift Workflow Commands

### Compile Single File
```sh
swiftc main.swift -o executableName
```

### Run Single File
```sh
./executableName
```

## Large Swift Workflow Commands

### Create Package
```sh
swift package init --type executable
```

### Run Package (ex tic-tac-toe)
```sh
swift run 
```
